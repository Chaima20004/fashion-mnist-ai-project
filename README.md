##  Descrizione

Progetto completo per il corso di Intelligenza Artificiale che implementa un classificatore di immagini Fashion-MNIST utilizzando una CNN con PyTorch. Il progetto include l'intero ciclo di sviluppo ML: analisi dati, addestramento modello, containerizzazione Docker, pipeline CI/CD automatizzata e interfaccia web interattiva.

##  Caratteristiche

- ** Modello CNN** con PyTorch per classificazione immagini
- ** EDA Completo** (Exploratory Data Analysis) su Fashion-MNIST
- ** Dockerizzazione** completa con 2 container (training + deployment)
- ** CI/CD Pipeline** con GitHub Actions (test automatici, linting, build)
- ** Interfaccia Web** con Gradio per demo interattiva
- ** Test Unitari** automatizzati con pytest
- ** Monitoraggio** training con salvataggio modelli e storia
