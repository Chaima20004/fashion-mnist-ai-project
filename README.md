Progetto completo per il corso di Intelligenza Artificiale che implementa un classificatore di immagini Fashion-MNIST con:

- **Docker** per containerizzazione
- **CI/CD** con GitHub Actions
- **EDA** completo (Exploratory Data Analysis)
- **Interfaccia web** con Gradio
